### Tool Shed

##### count (classes) 
Script to count classes in files or repos.

###### run
```bash
# setup venv, install reqs
count.py ../filename.py 
```

###### output
```bash
{
    "filename.py": 3
}
```

##### scrap (types) 
Script to remove type anotations from python files

###### run
```bash
# setup venv, install reqs
scrap.py ../filename.py 
```

###### output
```bash
done. see ../filename_scrapped.py 
```
